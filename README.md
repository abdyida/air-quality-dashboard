# 🌍 Air Quality Dashboard

This project is an interactive dashboard built using Streamlit and OpenAQ API.

## Features
- Real-time air quality data
- Filters (country, pollutant, date)
- Interactive charts

## Why I chose this dataset
I chose air quality data because it is an important global issue affecting public health. Using a live API makes the dashboard dynamic and useful.

## Run locally
streamlit run app.py
